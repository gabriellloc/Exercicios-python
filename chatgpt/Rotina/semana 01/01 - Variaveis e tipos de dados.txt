Variaveis, tipos de dados

Variáveis - São “caixas” onde você guarda valores.
    Exemplo:
        #nome = "João"
        #idade = 25
        #altura = 1.75

Dados possiveis:
    int - Serve para números inteiros
    float - Números com casas decimais
    str - Texto (string)
    bool - Booleano (True/False)

Atribuição de variáveis
type() - Voce utiliza para verificar o tipo da variavel.
    Exemplo: 
        print(type(idade))

input() - Voce utiliza para receber um valor do usuario.
    Exemplo:
        var = input("Qual o seu nome")
